## **Задачи на сегодня**
### **Шаг 1: Подготовка структуры**
1. **Создать папку проекта** со следующей структурой:
    - Файл со структурой `README.md` для описания проекта
    - Папка `src/` для хранения исходного кода
    - Папка `docs/` для документации 
    - Файл `.gitignore` для исключения ненужных файлов

### **Шаг 2: Настройка репозитория**
2. Инициализировать Git-репозиторий
    - `git init`
3. Настройка репозитория
    - Добавить все файлы в staging area `git add .`
4. Проверить статуc изменений 
    - `git status`
5. Создать первый коммит
    - `git commit -m "Название"`
### **Шаг 3: Работа с удалённым репозиторием**
4. Настройка удалённого репозитория
```
git remote add origin <url>
git push -u origin main
```
